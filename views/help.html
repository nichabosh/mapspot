üìç Getting Started with Mapbox

The Maproom project uses an open-source mapping platform called Mapbox. For the majority of this tutorial, we'll be working with a product of theirs known as Mapbox Studio, which is essentially like Photoshop for geographical data.

If you haven't already, navigate over to the Mapbox website and create a new account. All maps you create and upload will be associated with this account, so be sure to save your password somewhere safe!

üë®üèª‚Äçüíª Importing Your Data

For the purposes of this tutorial, I'm going to provide you with a practice dataset to work with. Though Mapbox accepts all different types of data, including KML, GPX, GeoJSON, Shapefile (zipped), CSV, and MBTiles, we're going to be using a zipped shapefile for this demonstration.

Once you've downloaded the attached data, navigate to the Mapbox homepage and click on your avatar in the top-right. A dropdown menu should appear with three main options: Account, Studio, and Documentation. Select Studio and, when the page loads, click the blue button labeled New Style.

At this point, you're free to choose whichever template you like. Mapbox should provide a preview of what the map will initially look like off to the right, and the data you eventually upload will be superimposed on top. For visibility's sake, I would encourage you to select either the light or dark variation of the Monochrome style, both of which can be seen in the images below.

Once you're in the editor view of the map, take a look at the toolbar located on the left side of the screen. Because we want to import our dataset as a new layer, select the Layers tab and click on the blue plus icon. This should open a window titled New Layer, which is where we'll import and style the river data downloaded previously. Now, click the Upload Data button, navigate to the zipped shapefile, select it, and press Submit. After the upload has finished processing, select the dataset, named River_Streams_Atlanta_Region, as the source and rename the layer to something more appropriate. In the image provided below, I chose to name it ATL Rivers and Streams.

üé® Styling Your Data

Now that we've uploaded our data and selected it as the source, let's first check to see what it looks like. On the map view, navigate to Atlanta, Georgia by typing the desired location in the hourglass searchbar (located in the top-right of the map view). Then, zoom in using the plus icon until the data is visible on the map. If you see a collection of black, veiny-looking lines, then congrats‚Äîyou've done everything correct so far! If not, go back and double check your steps.

Because we're looking to display waterways, perhaps black isn't the best color to use. In order to change this, select your layer in the leftmost panel and click the Style tab if you're still in the Select Data panel (or simply press "[" on your keyboard). Now, using the gradient provided under the Color tab, change the data to a more fitting color. Once you've finished that up, you should be all done customizing the data! Go ahead and publish your map by clicking the Publish button located in the top right corner of the screen.

üè† Uploading to Maproom

Now that we've published our map, we must also make sure it's publicly-accessible. In order to do so, navigate back to Mapbox Studio, click on the three vertical dots next to your map, and select Make Public. Next, click the three dots again, copy the Style Url, and navigate to the Maproom's upload portal.

Here, you can provide a unique name for your custom map using the Mapname input field. In this case, I'm going to give it a broad title, "Atlanta Data", because I might want to add more Atlanta-related layers to the map in the future. Next, paste the copied style url into the Style Url textbox and navigate back to Mapbox Account (in a new tab, of course) to access your account's public token. Scroll down on the page until you reach a section labeled Access Tokens‚Äîhere, you should see a Default Public Token of the form "pk.eyJ1IjoiYX". Copy this value, navigate back to the upload portal, paste the token in the Public Key input field, and click the Submit button.

If done correctly, you should be met with a message claiming that "Your map is pending approval". This means it has been successfully uploaded through our portal and will be displayed on the live site once we're able to verify and approve it, so just sit tight for the time being! In the meantime, feel free to check out some of the other approved maps.
